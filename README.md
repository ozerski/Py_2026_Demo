## Работа с git
### Локальный репозиторий 
Подключение проекта к git в терминале (обязательно из папки проекта):

- инициализация пустого репозитория (создаётся папка `.git`):
    **git init**
- просмотр текущего состояния проекта (н/обяз.)
(<span style="color: green;">отслеживаемые `Changes to be committed`</span>, 
игнорируемые (см. .gitignore),
<span style="color: brown;">не отслеживаемые `Untracked files`</span>): 
    **git status**
- исключение из отслеживаемых (например, настройки): указываем имя 
  файла/папки в файле `.gitignore` 
- индесация для последующего закоммичивания (точка = "все", либо названия 
  папок и файлов ч/з запятую):
    **git add .**
- создание коммита: **git commit -m "комментарии к коммиту"**
- просмотр коммитов (н/обяз.): **git log**
